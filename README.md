# Senda Animal

**Protegemos especies, restauramos futuros.**

Aplicación web desarrollada con Angular que simula la experiencia digital de un zoológico moderno. Permite a los usuarios explorar animales, gestionar favoritos, adquirir entradas e inscribirse en cursos y actividades según la estación del año.

## Dirección

(senda-animal.vercel.app)

## Funcionalidades

- Exploración de animales
- Sistema de favoritos
- Compra de entradas
- Inscripción a cursos
- Actividades organizadas por estación
- Registro y autenticación de usuarios
- Beneficios exclusivos para miembros

## Arquitectura

El proyecto está estructurado siguiendo buenas prácticas de Angular:

- Separación entre páginas y componentes reutilizables
- Uso de servicios para la lógica de negocio
- Implementación de guards de navegación
- Componentes configurables según el contexto (home vs páginas específicas)
- Organización orientada a escalabilidad y mantenimiento

## Tecnologías

- Framework: Angular
- Lenguaje: TypeScript

## Persistencia de datos

- Simulación de API mediante JSON local
- `localStorage`
- `IndexedDB`

## Patrones aplicados

- Repository pattern
- Factory pattern
- Strategy pattern
- Singleton

## Objetivo

Este proyecto forma parte de mi porfolio y tiene como objetivo profundizar en la arquitectura frontend, la experiencia de usuario y la reutilización de componentes en Angular.
